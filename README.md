# 🔓 VeilText Decoder

A lightweight text decoding tool built using HTML, CSS, and JavaScript. VeilText Decoder converts encoded strings back into original text using reverse transformation logic and base36 decoding, matching the VeilText Encoder system.

---

## 🚀 Features

- 🔓 Decode encoded text instantly  
- 🔢 Reverse ASCII transformation logic  
- 🔄 Base36 decoding support  
- ⚡ Fast and real-time output  
- 🎨 Hacker-style UI design  
- 💻 Lightweight and fast (no external libraries)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)

## 🎯 Usage

1. Paste encoded text into input box  
2. Click "Decode"  
3. Original text appears instantly  
4. Copy the decoded result  

---

## 🔐 How It Works

- Input is split using "." separator  
- Each value is converted from base36  
- Reverse transformation applied:  
  (value / 3) − 777  
- ASCII values converted back to characters
